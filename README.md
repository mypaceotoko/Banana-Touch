# 🍌 Banana Touch

> タップしてバナナを増やす、中毒性の高いクリッカーゲーム！

![Banana Touch](assets/banana.png)

---

## 🎮 ゲーム概要

**Banana Touch** は、バナナをタップして数を増やし、ショップでアップグレードを購入してさらに多くのバナナを集めるクリッカー / 放置ゲームです。

シンプルな操作でスマホでも楽しめます。

---

## 🕹️ 遊び方

1. **画面中央のバナナをタップ** してバナナを増やす
2. **ショップ** でアップグレードを購入して効率アップ
   - 👆 **クリック強化** — タップ1回で増えるバナナが増える
   - 🐒 **バナナモンキー** — 放置中も自動でバナナが増える
3. リロードしても進行状況は **自動保存** される
4. 放置中もバナナが増え続ける（最大24時間分）

---

## 🛠️ 技術構成

| ファイル     | 役割                        |
|--------------|-----------------------------|
| `index.html` | ゲームの基本構造・HTML骨組み |
| `style.css`  | モバイル優先デザイン・アニメーション |
| `game.js`    | ゲームロジック（状態管理・クリック・BPS） |
| `ui.js`      | DOM更新・エフェクト・ショップUI |
| `save.js`    | localStorage によるセーブ／ロード |
| `main.js`    | 初期化・各モジュール接続 |

- **Pure JavaScript**（フレームワーク不使用）
- **requestAnimationFrame** によるゲームループ
- **localStorage** によるデータ永続化
- モバイル優先レスポンシブデザイン

## 🚀 プレイ方法

ローカルでサーバーを立てるか、`index.html` を直接ブラウザで開いてください。

```bash
# Python でローカルサーバーを起動する場合
cd banana-clicker
python3 -m http.server 8080
# → http://localhost:8080 でアクセス
```

または VS Code の **Live Server** 拡張機能でも動作します。
